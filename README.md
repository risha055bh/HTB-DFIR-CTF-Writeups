# HTB DFIR CTF Writeups 🔍

A collection of Digital Forensics & Incident Response (DFIR) 
writeups from HackTheBox Sherlocks and CTF labs.

## What's Inside
Step-by-step solutions covering log analysis, memory forensics,
network forensics, malware analysis, and incident reconstruction
— with explained commands and real-world context.

## Skills Covered
- 🪵 Log Analysis (auth.log, wtmp, syslog)
- 🧠 Memory Forensics
- 🌐 Network Forensics (PCAP analysis)
- 🦠 Malware Analysis
- 🗺️ MITRE ATT&CK Mapping
- 🔎 Threat Hunting & Timeline Reconstruction

## Tools Used
Volatility, Wireshark, grep, 7z, Autopsy, and more.
